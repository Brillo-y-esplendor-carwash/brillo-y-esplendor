# 🚗 Brillo y Esplendor

Sitio web prototipo para un autolavado ubicado en Santiago, Chile. Permite a los clientes reservar horas de lavado y al personal administrativo gestionar las reservas desde un dashboard protegido.

## ✨ Funcionalidades

- Reserva de horas desde la página principal (formulario o chat IA)
- Selección de tipo de vehículo y servicio con precio estimado
- Dashboard de reservas con login para administradores
- Filtros por nombre, hora y fecha

## 🛠 Tecnologías

- HTML, CSS, JavaScript
- Node.js + Express
