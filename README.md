# 🚗 Brillo y Esplendor

Plataforma web prototipo desarrollada para la digitalización del autolavado **Brillo y Esplendor**.

El sistema permite a los clientes reservar servicios de lavado de vehículos de forma rápida y sencilla, mientras que el personal administrativo puede gestionar reservas y visualizar información relevante desde un dashboard protegido.

---

## ✨ Funcionalidades principales

### 👤 Clientes

* Reserva de horas online.
* Selección de tipo de vehículo y servicio.
* Cálculo automático de precios.
* Consulta de horarios de atención.
* Sistema de fidelización **Puntos Brillo**.
* Promociones y descuentos especiales.
* Asistente virtual integrado.
* Visualización de reseñas y calificaciones.
* Validación de fechas para evitar reservas en días ya transcurridos.

### 🛠 Administración

* Inicio de sesión protegido.
* Dashboard de gestión de reservas.
* Historial de reservas.
* Filtros por nombre, fecha y horario.
* Visualización de información de clientes y servicios.

## 🧪 Funcionalidades demostrativas del prototipo

Para representar algunas funcionalidades propuestas en el proyecto se incorporaron pantallas demostrativas:

### 📅 Agenda Inteligente

Simulación de:

* Disponibilidad de horarios.
* Detección automática de conflictos de reserva.
* Lista de espera digital.
* Notificación al cliente cuando se libera un turno.

Archivo:

```text
agenda-demo.html
```

### ⭐ Sistema de calificación mutua

Simulación de:

* Evaluación del servicio por parte del cliente.
* Evaluación del cliente por parte del autolavado.
* Comentarios internos y puntuación mediante estrellas.

Archivo:

```text
calificacion.html
```
